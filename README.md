# palaeoptera-phylogenomics
Scripts and workflow for palaeoptera project inlcuding downloading genomes and Busco genes. 
Contents:
-genome_download.py 
  python script to download genomes from NCBI
-download.sh 
  runs the python script
-run_busco_phylogeny.sh
  aligns, trims, concats, and creates text file for IQtree
